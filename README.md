![Wormhole showcase](Wormhole.gif)  

This project is a real-time, physically inspired wormhole ray tracer. It integrates simplified null geodesic trajectories through a prescribed traversable-wormhole-like geometry and visualises how two sky environments are gravitationally lensed through the throat.

Controls

Mouse	yaw / pitch view rotation
W / S	move along the wormhole coordinate l
A / D	horizontal visual offset
Q / E	camera roll
R	reset camera
1 / 2 / 3	switch sky rendering mode
ESC	quit
